# Changelog
All notable changes to VoiceCalendar AI will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-06-17

### Added
- Initial project structure with Clean Architecture + MVVM
- Voice recording with Android SpeechRecognizer API
- LLM-powered event extraction (configurable provider)
- Calendar integration via Android Calendar Provider
- Editable event confirmation dialog
- Material 3 theme with dynamic colors and dark mode
- Internationalization (en, es, ca)
- Settings screen for LLM configuration
- Upcoming events list
- Unit tests for domain and data layers
- GitHub Actions CI/CD pipeline
- Full project documentation
